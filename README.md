## Featured Engineering Projects

The projects in this portfolio represent a progression from operational automation to full-scale data engineering and AI-assisted analytics. While each was built to solve a different problem, they share the same philosophy: eliminate repetitive work, preserve data integrity, and build systems that empower people rather than replace them.

Across healthcare, community operations, and analytics, these projects demonstrate end-to-end ownership of technical solutions — from architectural design and infrastructure development to automation, validation, deployment, and continuous refinement.

### Featured Projects

* **AI Moderation & Privacy Infrastructure**
Designed an intelligent assistant for a large healthcare community capable of responding to moderator inquiries, enforcing community standards, and classifying sensitive content using structured decision logic. The system emphasizes privacy-by-design through carefully engineered data structures, strict guardrails, and the complete exclusion of personally identifiable information (PII).

* **AI Community Operations Platform**
Engineered a production-ready C# extension that transforms legacy software into an AI-assisted operational platform. The project combines software engineering, prompt engineering, dynamic personality management, UX design, and LLM orchestration to create configurable virtual community advocates capable of safe, context-aware interactions.

* **Autonomous Community Operations RPA**
Built a browser-based robotic process automation (RPA) pipeline that autonomously triages a community exceeding one million members. Operating through Selenium after public API access became unavailable, the system performs moderation, prioritization, and escalation while deferring only genuinely ambiguous cases to human operators through carefully defined programmatic guardrails.

* **CMS Data Engineering Pipeline**
Developed an automated ingestion pipeline for publicly available CMS datasets, transforming fragmented Excel-based source files into clean, normalized analytical datasets. The project includes data validation, transformation workflows, and downstream Power BI reporting designed for operational and reimbursement analysis.

* **Insight: CMS Part B Compliance Analytics Engine**
Architected a complete healthcare analytics platform that ingests, validates, models, and analyzes CMS Part B reimbursement data. Built on a dimensional star schema with analytical SQL views and AI-assisted narrative generation, the platform converts millions of raw records into executive-ready compliance insights while maintaining data quality throughout the ETL lifecycle.
